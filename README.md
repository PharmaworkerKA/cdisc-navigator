# CDISC Standards Navigator

## テーマ
CDISCデータ標準（SDTM、ADaM、Define-XML、CT等）の実務ノウハウを発信するブログ。規制当局への申請データ作成で現場が直面する課題と解決策を、具体的なコード例とともに解説する。

## ターゲット読者
- 製薬企業・CROのSASプログラマー/データマネージャー
- CDISC標準の導入・移行を担当するプロジェクトリーダー
- 臨床データ標準に関心のある薬事担当者

## 想定記事カテゴリ
- SDTM実装ガイド（ドメイン別マッピング、Supplemental Qualifiers）
- ADaM設計パターン（BDS、ADTTE、派生変数ロジック）
- Define-XML作成・バリデーション
- Pinnacle 21（OpenCDISC）対応とエラー解消
- CDISC最新アップデート・業界動向
